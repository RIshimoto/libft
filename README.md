## 概要
42Tokyo[^1]の課題の一つ。C言語の標準ライブラリ<strings.h>のライブラリ関数の一部を自作。

[^1]:https://42tokyo.jp/

### 実装した関数
- memset
- bzero
- memcpy
- memmove
- memchr
- memcmp
- strlen
- strlcpy
- strlcat
- strchr
- strrchr
- strnstr
- strncmp
- atoi
- isalpha
- isdigit
- isalnum
- isascii
- isprint
- toupper
- tolower
